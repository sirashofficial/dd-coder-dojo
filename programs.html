Our Learning Programs

Age-appropriate coding education paths for young innovators

Junior Ninjas Program

Introducing young minds to computational thinking through block-based programming and interactive games. Our youngest coders learn the fundamentals of programming logic through visual, drag-and-drop interfaces that make coding fun and accessible.

What You'll Learn

  • Scratch Programming Basics
  • Game Design Fundamentals
  • Logic Building Exercises
  • Creative Problem Solving
  • Basic Computer Navigation
  • Digital Storytelling

Class Schedule

Thursday: 14:00-15:00 | Friday: 13:00-15:00

Code Explorers Program

Transitioning to text-based programming with real-world web development projects. Students begin their journey into professional coding languages while building websites and simple applications.

What You'll Learn

  • HTML & CSS Foundations
  • JavaScript Basics
  • Python Programming
  • Simple Web Projects
  • Version Control (Git)
  • Database Basics

Class Schedule

Tuesday & Wednesday: 14:00-16:00

Tech Innovators Program

Advanced programming and physical computing for real-world problem solving. Our senior students work on complex projects that prepare them for tech careers and university computer science programs.

What You'll Learn

  • Full Stack Web Development
  • Advanced JavaScript & Python
  • Raspberry Pi Projects
  • Machine Learning Basics
  • Real-world App Development
  • Career Preparation & Portfolio Building

Class Schedule

Monday: 14:00-16:00
